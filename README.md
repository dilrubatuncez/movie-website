# 🎬 Flares — Movie & Series Streaming UI

A Netflix-style **movie & series streaming interface** built with vanilla **HTML, CSS & JavaScript**. Features horizontally scrolling content rows, a dark/light theme toggle, and multiple pages — a clean, responsive front-end project focused on layout, interactivity and UI detail.

> Built in 2022 as a student front-end project.

---

## ✨ Features
- 🎞️ **Horizontal movie carousels** — scroll through rows of titles with arrow controls (custom JavaScript)
- 🌗 **Dark / Light mode toggle** — animated theme switch
- 🧭 **Multi-page layout** — Home, show detail (Vikings) and Profile pages
- 📱 **Responsive design** — adapts to different screen sizes
- 🎨 **Modern UI** — Google Fonts, Font Awesome icons, poster grid

## 🛠️ Built With
- **HTML5** — semantic structure
- **CSS3** — layout, theming, responsive design
- **JavaScript** — carousel logic & theme toggle

## 📄 Pages
- `index.html` — Home page with scrolling movie/series rows
- `vikings.html` — Example show detail page
- `profile.html` — User profile page
